# 1
- Big-O 표기법은 시간복잡도를 나타내는 표현으로 최악의 경우에도 해당 시간 내에 문제를 해결할 수 있음을 의미한다.
- 예를 들어 O(n)의 경우 n개의 입력에 대해 선형비례하는 시간(kn) 내에 문제가 해결됨을 뜻한다.

# 2
- 시간복잡도: O(n^2)
- n에 대해 T(n) = T(n - 1) + 1, T(n - 1) = T(n - 2) + 1, ......, T(0) = 1이다
- 다시 말해, f(n)을 구하기 위해서는 0에서 n-1까지 n번 만큼의 연산이 필요하다.
- 이 때 각각의 연산에 필요한 계산량은 n에 비례한다
- 따라서 함수 f(n)의 총 계산량은 n * n으로 시간복잡도는 O(n^2)이다.
