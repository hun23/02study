3. 순회

3.1 중위순회
def inorder(v):
    if v:
        inorder(v.left)
        print(v)
        inorder(v.right)

'''
inorder(v) 정의:
    if v is not None 이라면
    v의 왼쪽 자식 정점을 방문하고
    v 자신을 방문해서 일을 처리하고
    v의 오른쪽 자식 정점을 방문한다.

이진 트리의 순회의 경우 전위, 중위, 후위 순회 모두 
왼쪽 자식노드를 방문 한 후 오른쪽 자식 노드를 방문하는데,
전위 순회는 v 본인 우선-왼쪽 오른쪽 자식 노드 방문 (= 부모 정점 먼저 방문)
중위 순회는 왼쪽 자식 노드-v-오른쪽 자식 노드 방문 (= 왼쪽-v-오른쪽 방문)
후위 순회는 왼쪽 자식 노드-오른쪽 자식 노드-v 방문 (= 자식 노드 먼저 방문)
로 이루어진다.
'''

3.2 트리의 방문 순서

pre_order(전위순회)  : A B C D E
in_order(중위순회)   : B A D C E
post_order(후위순회) : B D E C A
